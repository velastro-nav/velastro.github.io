# Velastro Navigation - GitHub Pages Website

A simple, elegant, and responsive GitHub Pages website built with HTML, CSS, and JavaScript.

## Features

- **Responsive Design**: Works seamlessly on desktop, tablet, and mobile devices
- **Smooth Navigation**: Smooth scrolling between sections
- **Modern UI**: Clean and contemporary design with gradient backgrounds
- **Interactive Elements**: Hover effects and scroll animations
- **Project Showcase**: Display your projects in a beautiful grid layout
- **Fast Loading**: Lightweight and optimized for performance

## File Structure

```
velastro.github.io/
├── index.html      # Main HTML file
├── styles.css      # Stylesheet
├── script.js       # JavaScript for interactivity
└── README.md       # This file
```

## Sections

1. **Header & Navigation** - Quick access to all sections
2. **Hero Section** - Eye-catching introduction

5. **Contact Section** - Call-to-action for inquiries
6. **Footer** - Copyright and information

## Customization

### Update Website Title
Edit the `<title>` tag in `index.html`:
```html
<title>Your Website Title</title>
```

### Change Colors
Modify the CSS variables in `styles.css`:
```css
:root {
    --primary-color: #3498db;
    --secondary-color: #2c3e50;
    --text-color: #333;
    --bg-color: #ecf0f1;
}
```

### Add Your Projects
Edit the projects section in `index.html` to add your own projects:
```html
<div class="project-card">
    <h3>Your Project Name</h3>
    <p>Your project description</p>
</div>
```
